# 알림 명세서 (Notification Spec)

> 모든 알림은 Slack Incoming Webhook 단일 채널을 통해 발송한다.
> 발송 트리거 / 타이밍 / 채널 구분 / 실패 처리 정책을 정의한다.

---

## 알림 채널 구성

| 채널 ID | 용도 | 대상 |
|---------|------|------|
| `#무제_공지` | 전체 공지 / 리마인더 | 무제 동호회 전체 회원 |
| Slack DM | 개인 알림 (수령 안내, 벌점, 이의 제기 결과) | 해당 회원 개인 |

- **채널 공지** (`#무제_공지`): 무제 동호회 전체 채널에 발송
- **채널 @멘션**: 슬랙 사용자 ID 미등록 회원은 이름(닉네임) 텍스트로 대체
- **개인 DM**: 슬랙 사용자 ID 필수. 미등록 회원은 개인 DM 건너뜀 — 채널 @멘션으로 대체
- 슬랙 사용자 ID 미등록 시 관리자 별도 알림 불필요 — 채널 @멘션 대체로 충분 (확정)

## 발송 트리거 목록

### NT-01: 독후감 마감 리마인더

- **트리거**: 독후감 마감일 기준 D-7 / D-3 / D-1
- **대상**: 해당 월 독후감 미제출 회원 전원
- **채널**: `#무제_공지` (전체 공지)
- **발송 시각**: 매일 오전 10:00 KST
- **타임존**: Asia/Seoul (KST, UTC+9)

**메시지 포맷 예시**

```
[무제] 독후감 마감 D-3 리마인더
이번 달 독후감 마감까지 3일 남았습니다.
아직 제출하지 않은 분들은 서비스에서 제출해 주세요.
마감일: 2026년 3월 31일 (화)
바로 가기: {독후감 제출 URL}
```

- D-1 메시지에는 미제출자 명단 포함 여부: [결정 필요: 명단 공개 / 비공개 (개인 DM으로만 발송)]

---

### NT-02: 독후감 마감 D-1 개인 DM

- **트리거**: 독후감 마감일 D-1
- **대상**: 해당 월 독후감 미제출 회원 개별
- **채널**: Slack DM
- **발송 시각**: 오전 10:00 KST (NT-01과 동시 발송)

**메시지 포맷 예시**

```
[무제] {이름}님, 독후감 마감이 내일입니다.
내일(2026년 3월 31일)까지 독후감을 제출해 주세요.
바로 가기: {독후감 제출 URL}
```

- **채널**: `#무제_공지`

바로 가기: {책 신청 URL}

### NT-04: 도서 도착 수령 알림 (인포 알림)

- **트리거**: 관리자가 도서 도착 처리를 완료한 시점
- **대상**: 해당 주문의 신청자 전원
- **채널**: Slack DM
- **발송 시각**: 상태 변경 즉시 (실시간)

**메시지 포맷 예시**

```
[무제] {이름}님이 신청한 도서가 도착했습니다.
도서: {도서명}
수령 기한: {도착일 + 7일}
수령 장소: [TBD: 수령 장소 필드 추가 여부 결정 필요]
바로 가기: {수령 확인 URL}
```

### NT-05: 인포 수령 기한 임박 알림

- **트리거**: 도서 도착일로부터 D+5 (수령 기한 2일 전)
- **대상**: 미수령 회원 개별
- **채널**: Slack DM
- **발송 시각**: 오전 10:00 KST

**메시지 포맷 예시**

```
[무제] {이름}님, 도서 수령 기한이 2일 남았습니다.
도서: {도서명}
수령 기한: {기한일}
수령 처리: {수령 확인 URL}
```

### NT-06: 벌점 부여 알림

- **트리거**: 관리자가 벌점을 부여한 시점
- **대상**: 벌점 부여 대상 회원
- **채널**: Slack DM
- **발송 시각**: 부여 즉시 (실시간)

**메시지 포맷 예시**

```
[무제] {이름}님에게 벌점이 부여되었습니다.
항목: {벌점 타입 한글명}
사유: {사유}
이의 제기: {이의 제기 URL}
```

---

### NT-07: 이의 제기 처리 결과 알림

- **트리거**: 관리자가 이의 제기 처리(수락 또는 기각)를 완료한 시점
- **대상**: 이의 제기 회원
- **채널**: Slack DM
- **발송 시각**: 처리 즉시 (실시간)

**메시지 포맷 예시 (ACCEPTED)**

```
[무제] 이의 제기가 수락되었습니다.
항목: {벌점 타입 한글명} / {날짜}
해당 벌점이 취소되었습니다.
```

**메시지 포맷 예시 (REJECTED)**

```
[무제] 이의 제기가 기각되었습니다.
항목: {벌점 타입 한글명} / {날짜}
사유: {관리자 입력 사유}
```

---

### NT-08: 촬영 일정 변경 알림

- **트리거**: 관리자가 촬영 일정을 수정하거나 삭제한 시점
- **대상**: 무제 동호회 전체 회원
- **채널**: `#무제_공지`
- **발송 시각**: 변경 즉시 (실시간)

**메시지 포맷 예시**

```
[무제] 촬영 일정이 변경되었습니다.
기존: {변경 전 일시} / {장소}
변경: {변경 후 일시} / {장소}
```

---

### NT-09: 도서 미반납 알림

- **트리거**: 대여 반납 기한 초과 후 매 7일마다 반복
- **대상**: 미반납 회원 개별
- **채널**: Slack DM
- **발송 시각**: 오전 10:00 KST

**메시지 포맷 예시**

```
[무제] {이름}님, 대여 중인 도서를 반납해 주세요.
도서: {도서명}
반납 기한: {기한일} (현재 {N}일 초과)
```

## 발송 타이밍 방식

- **즉시 발송** (NT-04, NT-06, NT-07, NT-08): 해당 이벤트 발생 즉시 발송
- **스케줄 발송** (NT-01, NT-02, NT-03, NT-05, NT-09): 매일 오전 10:00 KST 자동 발송
